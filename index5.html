Wow CSS Gradianets

Some Fancy Stuff

You can make them repeat.

Just type "repeating-radial-gradient(color1, color2 10%, color 3 15%);" you do have to had some boundries to the colors to indicate the size of the repeating gradianets.

Here I made the gradianets much smaller "repeating-linear-gradient(color1, color2 5%, color3 5%);"

You can make the gradient fade away.

Using alpha you can change the opacity of any of the colors, so you could make your gradient go completely transparent. "radial-gradient( color1, color2 (like rgba(0, 0, 0, 0)));"

Stack them up.

This has three gradients stacked on top of eachother to get this effect. It looks like this:

“background-image: linear-gradient(217deg, rgba(255, 0, 0, 0.8),rgba(255, 0, 0, 0) 70.71% ),

linear-gradient(127deg, rgba(0, 255, 0, 0.8), rgba(0, 255, 0, 0) 70.71%),

linear-gradient(336deg, rgba(0, 0, 255, 0.8), rgba(0, 0, 255, 0) 70.71%);”

as you can see, it uses hue and opacity.
