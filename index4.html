Wow CSS Gradianets

LINEAR

Default (top to bottom).

Just type "linear-gradient(color1, color2)"

You can do left to right.

Just type "linear-gradient(to right, color1, color2)"

You can do diagonals too!

Just type "linear-gradient(to bottom right, color1, color2)"

You can use degrees to rotate the gradient as well.

You’ll need to use degrees for this so type “linear-gradient(75deg, rgb(0, 255, 225), rgb(17, 17, 21));”
